Qué es

¿Qué cuenta este contador de números?

Dígitos individuales y secuencias numéricas del 0 al 9

Definición
Dígitos, enteros y decimales

Este contador distingue tres niveles: dígitos (cada carácter 0–9 individual), secuencias numéricas (grupos de dígitos consecutivos, con o sin separadores) y la clasificación en enteros (sin punto ni coma decimal) vs decimales (con . o , entre dígitos). También muestra el porcentaje que suponen los dígitos sobre todos los caracteres del texto.

Densidad
% de dígitos en el texto

Un texto periodístico normal tiene un 2–5% de dígitos. Un informe financiero puede superar el 15%. Este porcentaje ayuda a calibrar el balance entre texto narrativo y datos.

Privacidad
Análisis local sin servidores

Todo el procesamiento ocurre en tu navegador. Puedes analizar informes confidenciales con números financieros sensibles sin ningún riesgo.

Casos de uso

¿Cuándo necesitas contar números?

Informes financieros y estadísticos

Verifica que un informe contiene el número esperado de datos numéricos y que el porcentaje de dígitos es coherente con el tipo de documento.

Periodismo de datos

Los artículos de datos journalism tienen una alta densidad numérica. Este contador ayuda a calibrar el equilibrio entre cifras y texto narrativo para mantener la legibilidad.

Validación de formularios y datos

Al procesar datos extraídos de formularios o scraping, este contador ayuda a verificar que los campos numéricos contienen la cantidad correcta de dígitos antes del procesamiento.

Análisis de contratos y documentos legales

Los contratos contienen fechas, importes y referencias numéricas. Detecta rápidamente cuántas cifras hay en un contrato para revisar posibles errores u omisiones.

Educación y ejercicios matemáticos

Comprueba si un enunciado de problema matemático tiene el número correcto de datos numéricos antes de publicarlo o enviarlo como tarea.

Control de calidad editorial

Un texto con demasiados números seguidos puede resultar árido. El porcentaje de dígitos ayuda a detectar secciones con excesiva carga numérica y reescribirlas con texto explicativo.

Preguntas frecuentes

Un dígito es cada carácter individual del 0 al 9. Un número (o secuencia numérica) es un grupo de dígitos consecutivos que representa una cantidad. El texto «en 2024 se vendieron 1.250 unidades» tiene 8 dígitos (2,0,2,4,1,2,5,0) y 2 secuencias numéricas (2024 y 1.250).
Se considera decimal cualquier secuencia numérica que contenga un punto (.) o una coma (,) entre dígitos. Así, tanto «3.14» como «1.250,50» se detectan como decimales. Los números sin separador decimal (como «2024» o «1250») se clasifican como enteros.
