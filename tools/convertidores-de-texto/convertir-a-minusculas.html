Conversor de texto a minúsculas: normaliza cualquier texto al instante

El conversor de texto a minúsculas online transforma instantáneamente cualquier texto pasando todas las letras mayúsculas a su forma en minúscula. Funciona directamente en el navegador, sin instalar ninguna aplicación y sin enviar tu texto a ningún servidor: total privacidad. Compatible con el español completo: convierte correctamente tildes (ÁÉÍÓÚ → áéíóú), eñes (Ñ → ñ) y diéresis (Ü → ü).

Cómo funciona paso a paso

1

Pega o escribe

Introduce tu texto en el campo izquierdo. Puede ser un email en mayúsculas, texto de Word, un PDF escaneado o código.

2

Conversión automática

El resultado en minúsculas aparece al instante en el panel derecho, sin necesidad de pulsar ningún botón.

3

Copia con un clic

Pulsa "Copiar resultado" y el texto en minúsculas queda listo en tu portapapeles para pegarlo donde lo necesites.

Texto original
HOLA, ¿CÓMO ESTÁS? LA Ñ Y LAS TILDES: Á, É, Í, Ó, Ú.

En minúsculas
hola, ¿cómo estás? la ñ y las tildes: á, é, í, ó, ú.

¿Cuándo usar texto en minúsculas?

Normalizar textos de Word o PDF

Los documentos escaneados o copiados desde Word a veces tienen mayúsculas inconsistentes. Normaliza en un clic.

Preparar slugs para URLs

Las URLs deben estar en minúsculas. Convierte tu título a minúsculas y luego usa el generador de slug.

Limpiar datos para bases de datos

Estandariza campos de texto antes de importarlos para evitar duplicados por diferencias en capitalización.

Programación — toLowerCase()

Equivalente a str.toLowerCase() en JavaScript o str.lower() en Python. Útil para comparaciones case-insensitive.

Emails y formularios

Los emails se almacenan en minúsculas. Normaliza direcciones copiadas para evitar errores de autenticación.

SEO y metadatos

Las keywords y hashtags de redes sociales funcionan igual en mayúsculas o minúsculas, pero las minúsculas son el estándar para URLs y metadatos.

lowercase en diferentes lenguajes de programación

Lenguaje Función lowercase Ejemplo
JavaScript str.toLowerCase() "HOLA".toLowerCase() → "hola"
Python str.lower() "HOLA".lower() → "hola"
PHP strtolower($str) strtolower("HOLA") → "hola"
Java str.toLowerCase() "HOLA".toLowerCase() → "hola"
SQL LOWER(columna) SELECT LOWER(nombre) FROM tabla
