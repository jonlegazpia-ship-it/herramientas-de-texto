¿Qué es Base64?

Base64 es un sistema de codificación que convierte datos binarios (o texto) en una cadena de caracteres ASCII imprimibles usando un alfabeto de 64 caracteres (A–Z, a–z, 0–9, + y /). Se usa ampliamente para transmitir datos en contextos donde solo se permiten caracteres de texto: emails, URLs, APIs REST y almacenamiento de imágenes en HTML como data:image/png;base64,....

Base64 no es cifrado

Es importante entender que Base64 es codificación, no cifrado. Cualquier persona puede decodificar un string Base64 al instante. No uses Base64 para proteger información sensible; usa algoritmos de cifrado reales como AES o bcrypt para contraseñas.
