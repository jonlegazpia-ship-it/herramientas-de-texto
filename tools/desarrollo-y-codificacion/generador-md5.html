¿Qué es un hash MD5?

MD5 (Message-Digest Algorithm 5) es una función de hash criptográfica que convierte cualquier texto o archivo en una cadena hexadecimal de 32 caracteres (128 bits). Dos textos diferentes nunca producen el mismo hash (en la práctica), y cambiar un solo carácter del texto produce un hash completamente distinto.

¿Para qué se usa MD5 hoy en día?

Aunque MD5 ya no se recomienda para seguridad criptográfica (se han encontrado colisiones), sigue siendo muy útil para verificar la integridad de archivos descargados, generar identificadores únicos no críticos, o hacer checksums rápidos de datos. Para almacenar contraseñas usa bcrypt o Argon2.
