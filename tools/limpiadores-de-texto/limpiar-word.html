Los problemas que Word causa al publicar texto en web

Qué limpia

Reemplaza comillas tipográficas (‘ ’ “ ”) por comillas rectas, guiones largos (–—) por guiones normales y puntos suspensivos (…) por tres puntos simples.

Por qué importa

Los caracteres especiales de Word pueden romper scripts, causar errores en bases de datos y generar caracteres extraños en formularios y sistemas de email.

Cómo usarla

Copia el texto de Word o Google Docs y pégalo en el campo izquierdo. El resultado limpio aparece instantáneamente listo para copiar y usar.

Limpiador de texto de Word: elimina caracteres tipográficos problemáticos

Cuando copias texto de Microsoft Word o Google Docs y lo pegas en un sistema web, editor de código o base de datos, suelen aparecer caracteres extraños. Word y Google Docs reemplazan automáticamente las comillas rectas por comillas tipográficas (curly quotes), los guiones dobles por guiones largos y los tres puntos por el carácter especial …. Este limpiador convierte todos esos caracteres a sus equivalentes ASCII estándar, manteniendo las tildes y la ñ del español intactas.

Caracteres que se reemplazan

  • Comillas tipográficas: ‘ → ', ’ → ', “ → ", ” → "
  • Guiones largos: – (en dash) → -, — (em dash) → -
  • Puntos suspensivos: … → ...
  • Espacio de no ruptura:   → espacio normal
  • Viñetas y puntos especiales: • → *

¿Funciona también con Google Docs?

Sí. Google Docs usa los mismos caracteres Unicode especiales que Word. Esta herramienta los limpia igualmente. También funciona con texto copiado de LibreOffice, Pages (Mac) y cualquier editor de texto enriquecido que aplique autocorrección tipográfica.
