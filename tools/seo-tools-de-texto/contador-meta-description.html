Contador de meta description: optimiza tus snippets para Google

El contador de meta description online analiza en tiempo real la longitud de tu descripción SEO y te indica si está dentro del rango óptimo para Google. El límite de visualización es de aproximadamente 160 caracteres en escritorio y 120 en móvil. Textos más largos se truncan con puntos suspensivos, cortando potencialmente el mensaje clave de tu página.

Fórmula para una meta description perfecta

  • Incluye la keyword principal cerca del inicio: Google la pone en negrita cuando coincide con la búsqueda del usuario.
  • Describe el beneficio concreto: qué ganará el usuario al hacer clic. No qué es la página, sino qué obtendrá.
  • Incluye un CTA implícito: “Descubre”, “Aprende”, “Calcula”, “Compara” — verbos de acción que invitan al clic.
  • Longitud entre 120 y 160 caracteres: usa este contador para verificarlo antes de publicar.
  • Una por página: las meta descriptions duplicadas son señal negativa para Google.

¿Por qué una buena meta description mejora el SEO?

Aunque Google no usa la meta description como factor de ranking directo, sí impacta en el CTR (Click-Through Rate). Un CTR alto indica a Google que tu resultado es relevante para esa búsqueda, lo que puede mejorar tu posición a medio plazo. Optimizar la meta description es una de las acciones de SEO on-page con mayor retorno de inversión por el poco tiempo que requiere.
